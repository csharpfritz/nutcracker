# Nutcracker Holiday Display

A Blazor Server application for Raspberry Pi Zero that controls a holiday nutcracker decoration with a 32Ã—8 WS2812B LED matrix and synchronized music playback. Features real-time LED lightshows, Bluetooth audio, and a web-based control UI.

## Features
- LED matrix animations synchronized to music
- Blazor Server web interface for control
- Raspberry Pi GPIO and native LED library integration
- Bluetooth audio playback
- Easily extensible with new lightshows and patterns
